Uji Coba 6




type="image": Digunakan untuk memilih gambar. Elemen ini akan menampilkan gambar dengan atribut src yang diberikan.
input type="file": Digunakan untuk memilih file dari komputer pengguna. Elemen ini akan menampilkan sebuah dialog box untuk memilih file apabila di-click.
input type="hidden": Digunakan untuk menyimpan data yang tidak perlu ditampilkan ke pengguna. Elemen ini akan mengirimkan data ke server apabila form dikirim.

Uji Coba 7



type="number": Digunakan untuk memasukkan angka. Elemen ini memiliki atribut min dan max untuk mengatur batasan nilai yang dapat dimasukkan.
type="range": Digunakan untuk memilih nilai dari rentang yang ditentukan. Elemen ini memiliki atribut min dan max untuk mengatur batasan nilai yang dapat dipilih.

Uji Coba 8

type="search": Digunakan untuk memasukkan teks pencarian. Elemen ini mirip dengan type="text", tetapi tetapi memiliki fitur untuk menghapus teks yang diketik.
type="tel": Digunakan untuk memasukkan nomor telepon. Elemen ini akan memvalidasi apakah teks yang dimasukkan memiliki format nomor telepon yang benar.
